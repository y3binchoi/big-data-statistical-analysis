결측값과 이상점

결측값 처리

  • 제거
  • 대체

제거

  • 행 제거 = 관측값 제거 = 레코드 제거
    • 모든 행이 NA
    • 특정 변수가 NA
    • NA가 몇개 이상
  • 열 제거 = 변수 제거 = 필드 제거
    • dropna
    • 특정 변수 값을 NaN으로 처리

대체

수치자료이고 다른 변수와 관련이 없는 경우 자기 참조

  • 대체값 직접 지정

    - 평균, 중앙값, 최빈값 문자일 땐 최빈값 - 무작위(핫덱)

  • 전/후 관측값으로 대체

    : 관측값의 순서가 시간순인 경우

다른 변수와 관련 있음

  • 회귀분석의 예측값 (+단차 약간의 변동을 추가해서 넣을 수도)

## Accuracy

혼동행렬(Confusion matrix)

분류 \ 실제 | Positive | Negative
---------|--------|--------
Pasitive |  TP  | FP
Negative |  FN  | TN

- 정확도(accuracy) = (TP+TN)/(TP+FN+FP+TN)
- 정밀도(precision) = TP/(TP+FP)
- 재현율(recall, sensitivity) = TP/(TP+FN)
- F1 = 2 (precision x recall)/(precision+recall)
    - precision과 recall의 조화평균, imbalanced 구조인 경우 많이 사용
- macro: label별 각 합의 평균
- micro: 전체평균

선형회귀 & 로지스틱 회귀 pandas, numpy, matplotlib.pyplot, sklearn.linear_model, statsmodels.api

선형회귀는 종속변수와 독립변수 사이의 선형적 관계를 가정한 회귀 분석입니다. 예측값과 실제값 사이의 차이를 최소화하는데 중점을 두어 계수를 추정합니다. 또한, 회귀 계수를 이용하여 독립변수와 종속변수 사이의 선형 방정식을 구성하고, 이를 통해 새로운 독립변수의 값을 입력하여 종속변수를 예측할 수 있습니다.

coef_는 기울기를 나타내며, intercept_는 절편을 나타냅니다

로지스틱 회귀는 종속 변수와 독립 변수의 관계를 모델링하는 데 사용되는 회귀 분석입니다. 이는 일반적으로 이항 분류 문제를 해결하는 데 사용됩니다. 로지스틱 회귀 모델에서는 로짓 함수를 사용하여 종속 변수가 특정 값을 갖을 확률을 추정합니다.
from statsmodels.formula.api import ols

model = ols('y ~ x1 + x2 + x3', data=df).fit()
predictions = model.predict(df[['x1', 'x2', 'x3']])

print_model = model.summary()
print(print_model)

위 코드는 y를 x1, x2, x3의 선형 조합으로 모델링하는 OLS 회귀 분석을 수행하는 코드입니다. data는 데이터프레임을, formula는 종속변수와 독립변수 간의 관계식을 문자열로 나타냅니다. fit() 메소드를 사용하여 모델을 피팅하고, summary() 메소드를 사용하여 결과를 출력합니다.

OLS 회귀는 "ordinary least squares regression"의 약어로, 선형 회귀 모델의 파라미터를 추정하는 방법 중 하나입니다. 이 방법에서는 잔차 제곱합을 최소화하여 데이터에 가장 잘 맞는 선을 찾아냅니다. OLS 회귀는 종속 변수와 하나 이상의 독립 변수 간의 관계를 모델링하는 데 사용되며, 통계 및 기계 학습에서 일반적으로 사용됩니다. statsmodels 및 scikit-learn과 같은 여러 소프트웨어 패키지에서 구현되어 있습니다.

LASSO와 Ridge regression은 선형 회귀의 일종으로, 회귀 모델의 과적합을 방지하는 방법입니다. 이 두 방법은 일반적으로 변수 선택이나 차원 축소에 사용됩니다.

LASSO(regularization path with L1 penalty)는 L1 패널티를 사용하여 회귀 계수를 규제합니다. L1 패널티는 계수를 0으로 만들 수 있으므로 모델에서 변수를 선택하는 능력을 가집니다. 이 방법은 변수 선택 기능을 수행하므로, 데이터 세트에서 중요하지 않은 변수를 제거하고 모델을 단순화할 수 있습니다.

Ridge(regularization path with L2 penalty)는 L2 패널티를 사용하여 회귀 계수를 규제합니다. 이는 모든 계수를 축소하지만, 0으로 만들지는 않습니다. Ridge regression은 LASSO의 단점인 변수 선택 능력이 없지만, 변수 간의 상관 관계가 높은 경우에 유용합니다.

두 방법은 모두 회귀 계수를 축소하여 과적합을 방지하며, 차이점은 규제 방법입니다. LASSO는 계수를 0으로 만드는 능력이 있어 변수 선택 기능을 수행합니다. Ridge는 L2 패널티를 사용하여 모든 계수를 축소하지만, 0으로 만들지는 않습니다. 따라서 Ridge는 변수 간 상관 관계가 높은 경우에 더 유용할 수 있습니다.

다범주 로짓 모형(multinomial logistic regression)은 종속 변수가 2개 이상의 범주가 있는 경우에 사용되는 로지스틱 회귀 분석입니다. 이는 일반적으로 다중 분류 문제를 해결하는 데 사용됩니다. 로지스틱 회귀 모델에서는 로짓 함수를 사용하여 종속 변수가 특정 값(보통 0 또는 1)을 갖을 확률을 추정합니다. 하지만 다범주 로짓 모형에서는 종속 변수가 2개 이상의 범주를 가지므로, 이를 표현하기 위해 소프트맥스(softmax) 함수를 사용합니다. 소프트맥스 함수는 범주별로 확률을 계산하고, 이를 합치면 1이 되도록 정규화합니다. 다범주 로짓 모형은 로지스틱 회귀 분석과 마찬가지로 독립 변수와 종속 변수 간의 관계를 모델링하는 데 사용됩니다.

로지스틱 회귀 모형과 다범주 로짓 모형은 모두 분류 문제에 사용되는 통계 모델입니다. 그러나 두 모델 사이에는 몇 가지 차이점이 있습니다.

  1. 종속 변수의 유형:
    • 로지스틱 회귀 모형: 이진 분류 문제에 적용됩니다. 종속 변수는 두 개의 클래스 중 하나에 속하는 경우에 사용됩니다.
    • 다범주 로짓 모형: 다중 분류 문제에 적용됩니다. 종속 변수는 세 개 이상의 클래스 중 하나에 속하는 경우에 사용됩니다.
  1. 확률 분포:
    • 로지스틱 회귀 모형: 로지스틱 회귀는 종속 변수의 확률 분포로 이항 분포를 가정합니다. 종속 변수의 값을 0 또는 1로 예측하며, 로지스틱 함수를 사용하여 이진 분류를 수행합니다.
    • 다범주 로짓 모형: 다범주 로짓 모형은 다항 로지스틱 분포를 가정합니다. 종속 변수의 값은 다중 클래스 중 하나가 될 확률을 나타내는 벡터입니다. 소프트맥스 함수를 사용하여 다중 클래스 분류를 수행합니다.
  1. 모수 추정 방법:
    • 로지스틱 회귀 모형: 로지스틱 회귀 모형은 최대 가능도 추정 방법을 사용하여 모수를 추정합니다. 일반적으로 경사 하강법과 같은 반복적인 최적화 알고리즘을 사용하여 최적의 모수를 찾습니다.
    • 다범주 로짓 모형: 다범주 로짓 모형은 로그-우도를 최대화하는 방법으로 모수를 추정합니다. 로그-우도 함수는 모든 클래스에 대한 확률의 곱으로 정의됩니다. 최적화 알고리즘은 일반적으로 다변수 경사 하강법 또는 이에 대한 변형을 사용합니다.
